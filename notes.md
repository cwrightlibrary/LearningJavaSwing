## Example project structure:

```css
MySwingApp/
├── src/
│   └── main/
│       ├── java/

│       │       └── yourcompany/

│       │               ├── gui/

│       └── resources/

├── out/ (generated automatically by IntelliJ)
├── .idea/ (IntelliJ project config files)
├── MySwingApp.iml
└── README.md
```
